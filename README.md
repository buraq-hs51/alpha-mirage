# 🌌 Alpha Mirage

> Interactive portfolio website for Quantitative Developers & Financial Engineers

[![Deploy to GitHub Pages](https://github.com/SH-Ahmed17/alpha-mirage/actions/workflows/deploy.yml/badge.svg)](https://github.com/SH-Ahmed17/alpha-mirage/actions/workflows/deploy.yml)

## ✨ Features

- **Interactive Cursor Effects** - Multi-layer glow trail with particle effects
- **Matrix Rain** - Falling mathematical symbols and Greek letters
- **Live Stock Ticker** - Scrolling stock prices animation
- **Candlestick Charts** - Animated financial visualizations
- **Dark Trading Terminal Theme** - Professional quant aesthetic

## 🛠️ Tech Stack

- React 19 + TypeScript
- Vite 7.3
- Tailwind CSS 4
- Framer Motion
- Radix UI Components

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
